
This is a easy change based on BGP simple

The user needs to do nothing changing comparing to BGP simple, but user could input raw binary data as an input to feed to BGP speaker

BGP DUMP is the intermediate program that program would call.

There are two example routing information input file offered for user to test
myroutes is in TABLE_DUMP_V2 format
bview.20090820.2359.gz is a binary format BGP raw data
